## 들어가며

[Ubuntu Linux에 zsh을 설치하고, default shell을 zsh로 바꾸자.](install-zsh-on-ubuntu-linux.md)에서 Ubuntu Linux의 default shell을 bash에서 zsh로 변경했다.  
변경을 했지만은, 아무런 theme가 적용되지 않은 zsh은 너무 못생겼다.  
그래서 `ohmyzsh`을 설치하여, shell을 꾸며보자.
